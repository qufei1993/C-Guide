# 数组

数组，是有序的元素序列，用于存储多个相同数据类型的集合。在 C 语言中， 数组属于构造数据类型。一个数组可以分解为多个数组元素，这些数组元素可以是基本数据类型或是构造类型。因此按数组元素的类型不同，数组又可分为数值数组、字符数组、指针数组、结构数组等各种类别。

## 数组声明格式

```c
数据类型 数组名称[长度];
数据类型 数组名称[长度] = 元素;
数据类型 数组名称[] = 元素;
```

所有数组元素以 0 开始，数组元素个数不能大于数组初始化长度。

## 一维数组

一维数组是最简单的数组，其逻辑结构是线性表。要使用一维数组，需经过定义、初始化和应用等过程。

声明一个长度为 10 的数组元素，为其赋值

```c
#include <stdio.h>
int main()
{
    int arr[10];
    int len = sizeof(arr) / sizeof(int);
    
    for (int i=0; i<len; i++) {
        arr[i] = i * 2;
    }
    
    printf("%d\n", arr[3]); // 6
    printf("%d\n", len); // 10 
    return 0;
}
```

初始化一个一维数组，求最大值

```c
#include <stdio.h>
int main()
{
    int arr[] = {10, 8, 16, 4, 12};
    int len = sizeof(arr) / sizeof(int);
    int max = 0;
    
    for (int i=0; i<len; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    
    printf("%d\n", max); // 10 
    return 0;
}
```

## 多维数组

C 语言支持多维数组，最简单的形式则为二维数组，例如声明一个 x 行 y 列的数组如下所示：

```
数据类型 数组名称[x][y];
```

初始化一个三行三列的数组如下所示：

```c
int arr[3][3] = {  
    {0, 1, 2} ,   /*  初始化索引号为 0 的行 */
    {3, 4, 5} ,   /*  初始化索引号为 1 的行 */
    {6, 7, 8} ,   /*  初始化索引号为 2 的行 */
};
```

多维数组遍历，输出偶数

```c
#include <stdio.h>
int main()
{
    int arr[3][3] = {  
        {0, 1, 2} ,   /*  初始化索引号为 0 的行 */
        {3, 4, 5} ,   /*  初始化索引号为 1 的行 */
        {6, 7, 8} ,   /*  初始化索引号为 2 的行 */
    };

    for (int i=0; i<3; i++) {
        for (int j=0; j<3; j++) {
            int current = arr[i][j];
            
            if (current % 2 == 0) {
                printf("%d ", current); // 0 2 4 6 8 
            }
        }
    }
    
    return 0;
}
```
